# Multimodal Video RAG System

A Retrieval-Augmented Generation (RAG) system for video understanding.

## Features

- Video Segmentation
- Whisper Speech Transcription
- PANNs Audio Event Detection
- Vision-Language Analysis
- ChromaDB Vector Database
- Semantic Search
- Groq-powered Question Answering

## Pipeline

Video Upload
↓
Video Segmentation
↓
Whisper Transcription
↓
PANNs Audio Analysis
↓
Visual Scene Analysis
↓
Unified Records
↓
Embeddings
↓
ChromaDB Storage
↓
Semantic Retrieval
↓
LLM Answer Generation

## Example Questions

- Who is speaking in the video?
- What is the main discussion?
- How many people are present?
- What sounds occur in the video?
